### Instrucciones de Configuración

1. **Requisitos Previos**: Asegúrate de tener instalado Node.js y npm en tu máquina.
2. **Clonación del Repositorio**: Usa el siguiente comando para clonar el repositorio:
   ```bash
   git clone https://github.com/MemozMultimedia/formularioyamb.git
   ```
3. **Instalación de Dependencias**: Navega al directorio del proyecto y ejecuta:
   ```bash
   npm install
   ```
4. **Configuración del Entorno**: Crea un archivo `.env` con las variables de entorno necesarias para el funcionamiento del proyecto.

### Guía de Uso

1. **Ejecutar la Aplicación**: Después de instalar las dependencias, puedes iniciar la aplicación con:
